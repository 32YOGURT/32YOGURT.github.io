# Quartz 블로그 시작하기

안녕하세요! 이것은 테스트용 첫 번째 블로그 포스트입니다.

## Quartz란?

Quartz는 마크다운으로 작성한 콘텐츠를 정적 웹사이트로 변환해주는 프레임워크입니다.

### 주요 특징

- ⚡ **빠른 속도**: 정적 사이트 생성으로 빠른 로딩
- 🎨 **커스터마이징**: 자유로운 스타일 변경
- 📝 **마크다운 지원**: 익숙한 마크다운 문법
- 🔗 **위키링크**: `[[링크]]` 형태의 내부 링크 지원

## 코드 예시

```typescript
const hello = (name: string) => {
  console.log(`Hello, ${name}!`);
};

hello("32YOGURT");
```
